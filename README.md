## 什麼是Iphone
就是一台蘋果手機，但沒有蘋果味

## Iphone的誕生
因為賈伯斯擔心有iPod功能的手機，會侵蝕iPod市場，但現有市場卻已被模仿iPod設計的MP3廠商侵蝕而全球手機市場也正在爆發性成長，不做手機等於放棄市場。直到2004年賈伯斯才同意做手機。
